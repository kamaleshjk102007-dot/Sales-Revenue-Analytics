📊 Sales & Revenue Dashboard

📁 Dataset Structure

The dashboard expects data in the following format:

Column	Description
Date	Order Date
Product	Product Name
Category	Product Category
Region	Sales Region
Customer	Customer Name
Salesperson	Sales Representative
Qty	Quantity Sold
Revenue	Total Revenue
Profit	Total Profit
